## **1\. 产品概述 (Product Overview)**

### **1.1 背景与愿景**

* **背景**：机器人行业面临硬件普及与需求定制化的矛盾。行业缺乏一款能将“模糊需求”转化为“严谨方案”的中间层工具。  
* **愿景**：打造机器人行业的 **Figma**。让从业者（销售/PM/创业者）通过标准化流程，设计出符合工程逻辑的机器人产品，并自动生成商业交付物。  
* **核心价值**：**IPD 流程工具化** \+ **物理逻辑可视化** \+ **交付物标准化**。

### **1.2 目标用户**

* **核心用户**：机器人公司销售、产品经理、解决方案工程师。  
* **长尾用户**：机器人创业者、集成商。

### **1.3 产品形态**

* **Web SaaS 平台**：无须安装，浏览器即用。  
* **核心能力**：3D 拖拽设计、实时物理校验、自动生成文档（PDF/Word）。

## **2\. 核心业务流程 (Core Workflow)**

本产品遵循 **“隐形 IPD（集成产品开发）”** 流程：

1. **需求定义 (ODD)**：用户选择场景 \-\> 系统锁定物理约束（坡度、路面、温湿度）。  
2. **架构设计 (Layout)**：3D 拖拽组件 \-\> 系统自动吸附并建立拓扑关系。  
3. **逻辑校验 (Validation)**：实时计算功率/载重 \-\> 红绿灯反馈风险（**必须通过校验才可交付**）。  
4. **价值分析 (Business)**：输入人力成本 \-\> 系统计算 ROI 回本周期。  
5. **成果交付 (Export)**：生成标准报价单 (PDF) 或 可行性分析报告 (Word)。

## **3\. 功能需求说明 (Functional Requirements)**

### **3.1 模块一：场景化启动器 (Scenario Starter)**

**目标**：通过 ODD（运行设计域）预设，确保设计的起点是严谨的。

| ID | 功能点 | 详细描述 | 优先级 |
| :---- | :---- | :---- | :---- |
| F1.1 | **场景超市** | 提供卡片式入口：室外巡检、室内配送、类人服务、复合机器人。 | P0 |
| F1.2 | **智能预设** | 选中场景后，自动写入 ODD 参数： \- 室外：爬坡=15°，防水=IP54，越障=5cm。 \- 室内：爬坡=5°，路面=平滑。 *注：用户可在设置中微调，但会有风险提示。* | P0 |

### **3.2 模块二：3D 可视化编辑器 (Robo-Canvas)**

**目标**：所见即所得的拼搭体验。

| ID | 功能点 | 详细描述 | 优先级 |
| :---- | :---- | :---- | :---- |
| F2.1 | **组件库面板** | 左侧悬浮面板，分类展示：底盘、感知、执行、能源、算力。 | P0 |
| F2.2 | **智能吸附** | 拖拽组件时，自动检测底盘预设插槽（Socket），距离 \< 0.2m 自动吸附并对齐法线。 | P0 |
| F2.3 | **参数化变形** | 拖入通用模型时，根据数据库中的 specs.dims 自动缩放模型尺寸。 | P0 |
| F2.4 | **双向绑定** | 3D 画布与属性面板实时同步。 | P0 |
| F2.5 | **可视化辅助** | \- **FOV 视锥**：显示传感器覆盖范围。 \- **尺寸标注**：显示整机长宽高包围盒。 | P1 |

### **3.3 模块三：逻辑与校验引擎 (Logic Engine)**

**目标**：**严谨的物理裁判**。不提供“后门”，倒逼用户优化设计。

| ID | 功能点 | 详细描述 | 优先级 |
| :---- | :---- | :---- | :---- |
| F3.1 | **红绿灯校验** | 实时计算物理公式： 1\. 总功率 vs 电池最大输出 2\. 总重 vs 底盘载重 3\. 爬坡需求 vs 底盘爬坡能力 🔴 **红灯**：存在硬伤，**禁止导出交付物**（或导出带警告水印）。 🟢 **绿灯**：设计合理，允许交付。 | P0 |
| F3.2 | **风险雷达图** | 实时生成五维雷达图：通过性、续航、负载、智能、成本。 | P1 |
| F3.3 | **实时 BOM** | 显示组件清单及**市场参考总价**。支持一键“高/低配切换”。 | P0 |

### **3.4 模块四：商业价值分析 (Business Value)**

**目标**：辅助销售进行价值传递。

| ID | 功能点 | 详细描述 | 优先级 |
| :---- | :---- | :---- | :---- |
| F4.1 | **ROI 计算器** | 输入：替代人数、单人月薪、运维费率。 输出：回本周期（月）、5年 TCO 对比图表。 | P1 |
| F4.2 | **3D 批注** | 允许用户在 3D 模型上添加“锚点注释”（如：“此处需加防撞条”），用于记录非标需求。 | P1 |

### **3.5 模块五：交付物导出系统 (Export System)**

**目标**：分级交付，满足不同场景。

| ID | 功能点 | 详细描述 | 优先级 |
| :---- | :---- | :---- | :---- |
| F5.1 | **3D 截图引擎** | 一键获取当前 3D 视角的 4K 高清透明背景 PNG。 | P0 |
| F5.2 | **PDF 报价单** | **客户端生成**。内容：封面图、配置清单表格、市场参考总价、签字栏。 | P0 |
| F5.3 | **Word 咨询报告** | **客户端生成**。内容：可行性分析、项目备注、ROI 图表、风险评估。 *注：生成的 Word 预留了空白段落供用户后续自行润色。* | P0 |

## **4\. 非功能性需求**

* **性能**：3D 编辑器首屏加载 \< 3秒，运行帧率 60 FPS。  
* **兼容性**：Chrome / Edge / Safari (Desktop \+ iPad)。  
* **数据安全**：所有项目数据加密存储，支持用户导出离线备份。
